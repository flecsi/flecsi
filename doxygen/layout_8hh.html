enum flecsi::data::layout : size_t

A FleCSI layout provides a specific interface for different logical data layouts, e.g., dense vs. sparse. The actual data layout is implementation-dependent.

Enumerator
raw 

Uninitialized memory with no objects constructed or destroyed.

single 

Access to the single element of an array.

sparse 

Each element is a map with integer keys.

particle 

Unordered elements are added/removed up to a maximum number.
