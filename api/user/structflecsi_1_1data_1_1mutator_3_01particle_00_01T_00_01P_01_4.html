Detailed Description

template<class T, Privileges P>
struct flecsi::data::mutator< particle, T, P >

Mutator for particle fields.

This class is supported for GPU execution however it is not thread-safe. Iterators are invalidated only if their particle is removed.

Template Parameters
Pif write-only, all particles are discarded
