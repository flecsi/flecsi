Detailed Description

template<class R, typename T>
struct flecsi::data::reduction_accessor< R, T >

Accessor for computing reductions.

This class is supported for GPU execution. Name via field::reduction. Usable only with the global topology. Pass a normal field_reference. The previous field value contributes to the result.
