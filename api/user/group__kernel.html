Detailed Description

Local concurrent operations.

If Kokkos is not available, they simply execute serially. To avoid unnecessary copies, one needs to pass a view since the ranges provided by the user are copied.

This function is a wrapper for Kokkos::parallel_for that has been adapted to work with random access ranges common in FleCSI topologies.

In particular, this function invokes a map from the normal kernel index space to the FleCSI index space, which may require indirection.
