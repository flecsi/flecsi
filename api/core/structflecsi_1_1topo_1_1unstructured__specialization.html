Connectivity information to store.

The format is

list<from<cells, to<vertices, edges>>>,
from<...>, ...>

where each leaf is an index_space.

◆ entity_lists

using flecsi::topo::unstructured_specialization::entity_lists = list<>

Special entity lists to store.

The format is

list<entity<edges, has<dirichlet, neumann>>,
entity<...>, ...>

where each has associates entity_list values with an index_space.
