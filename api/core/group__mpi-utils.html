One-to-All (variable) communication pattern.

This function uses the FleCSI serialization interface with a packing callable object to communicate data from the root rank (0) to all other ranks.

Template Parameters
FThe packing functor type with signature (rank, size).
Parameters
ffunction object, invoked only on rank 0 and in recipient order
commAn MPI communicator.
Returns
For all ranks besides the root rank (0), the communicated data. For the root rank (0), the callable object applied for the root rank (0) and size.
