Detailed Description

template<class KEY, class TYPE, class HASH>
struct flecsi::util::hashtable< KEY, TYPE, HASH >

hashtable implementation.

This hashtable is based on span as 1D array and implement the method of displacement when there is a collision, placing the element at a distance of "modulo_". In this current implementation the neutral key is the default constructor of the object :key_t{}, 0 for numeric types.
