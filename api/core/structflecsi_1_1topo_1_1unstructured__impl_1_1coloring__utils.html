Detailed Description

template<typename MD>
struct flecsi::topo::unstructured_impl::coloring_utils< MD >

The coloring_utils interface provides utility methods for generating colorings of unstructured input meshes.

The member functions in this interface modify the internal state of the object to add or create coloring information. Once the requisite methods have been invoked, the coloring object can be accessed via the generate method.

Template Parameters
MDmesh definition, following mesh_definition; usually deduced from constructor argument

Build auxiliary entities for the given entity kind.

Parameters
kindThe mesh definition entity kind of the auxiliary to build.
hThe heuristic to use to assign colors to the newly-created auxiliaries. Specifying vertex will assign each new auxiliary the color of the lowest-ranked vertex that is connected to the auxiliary. Specifying cell will do the same, but with the lowest-ranked cell.
