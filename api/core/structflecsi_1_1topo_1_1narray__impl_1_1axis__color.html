◆ extended()

util::id extended ( ) const

The extended entities, i.e., the logical entities including boundary padding.

The boundary depth can be computed like: boundary_depth_low = logical<0>() - extended<0>(); boundary_depth_high = extended<1>() - logical<1>(); The ghost depth can be computed like: halo_depth_low = extended<0>(); halo_depth_high = extent() - extended<1>(); This function is supported for GPU execution.
